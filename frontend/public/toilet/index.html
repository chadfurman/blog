CSS Toilet Animation

This is a pure CSS toilet animation you can embed in your own website or blog.

Hover over the toilet to see the animation

How to Embed

Simply copy the following code and paste it into your website or blog:

<div style="position:relative;width:100%;max-width:500px;height:0;padding-bottom:75%;margin:0 auto;">
  <iframe src="https://chads.website/toilet/toilet.html" 
    style="position:absolute;top:0;left:0;width:100%;height:100%;border:none;" 
    frameborder="0" loading="lazy" allowtransparency="true"></iframe>
</div>
Note: Replace YOUR_WEBSITE_URL with your actual website URL where this toilet animation is hosted.
